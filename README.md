# Dhana siri Koppisetti Portfolio

A modern, responsive personal portfolio website built with React, Vite, and Framer Motion.

## Tech Stack

- React
- Vite
- Framer Motion
- CSS with Google Fonts

## Features

- Fully responsive layout for mobile, tablet, and desktop
- Sticky navigation with mobile menu
- Parallax decorative elements
- Scroll-triggered section and card animations
- Reduced-motion support for accessibility
- SEO-ready document head metadata

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

4. Preview the production build:

   ```bash
   npm run preview
   ```

## Deployment

Deploy the site with Vercel or Netlify after pushing the repository to GitHub. Replace the placeholder social and project links in `src/App.jsx` with your live profiles and real project repositories before submission.
